# What Would You Choose? by Ansa

A simple and fun choice-based game for kids, teens, adults, and families.

## Features

* 🎮 Single-player gameplay
* 👨‍👩‍👧‍👦 Categories for Kids, Teens, Adults, and Family
* 🤔 Hundreds of interesting dilemmas
* 📱 Fully responsive design
* ✨ Clean and modern interface
* 🌙 Smooth user experience

## How to Play

1. Select a category.
2. Enter player names.
3. Read the dilemma.
4. Choose between the two options.
5. Continue playing and have fun discussing your choices.
